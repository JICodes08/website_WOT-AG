How to Use

After including the notification.js script, you can use these methods:

Basic Usage:

// Show a simple notification window.notificationBanner.show('Your message here'); // Show with type window.notificationBanner.showInfo('Information message'); window.notificationBanner.showSuccess('Success message'); window.notificationBanner.showWarning('Warning message'); window.notificationBanner.showError('Error message');

Advanced Usage:

// Notification Banners (overlay, temporary) window.notificationBanner.show('Message', 'info', 5000); window.notificationBanner.show('Persistent message', 'warning', 0, true); window.notificationBanner.hide(); // Permanent Banners (static, doesn't overlap content) window.notificationBanner.showPermanent('Promotion message', 'promotion', true); window.notificationBanner.showPermanentInfo('Information banner'); window.notificationBanner.hidePermanent();

Common Use Cases:

// Permanent promotion banner window.notificationBanner.showPermanentPromotion('25% off window tinting - Limited time!', true); // Permanent business announcement window.notificationBanner.showPermanentInfo('Now offering mobile service!', true); // Temporary welcome notification window.notificationBanner.showSuccess('Welcome! Thanks for visiting.', 5000); // Holiday hours (permanent until dismissed) window.notificationBanner.showPermanentWarning('Holiday Schedule: Closed Dec 25-26', true);
